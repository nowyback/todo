Todoodle v0.9.1

A minimalist task manager designed specifically for developers who want a clean, efficient way to manage their todos without unnecessary complexity.

What's New
- Custom branded Windows installer with proper icon integration
- Streamlined project structure for easier maintenance
- Mobile and web app support through PWA technology

Features
- Clean, minimalist interface with dark theme
- Individual task progress tracking with color-coded sliders
- Custom categories for organizing your work
- Notes functionality for detailed task information
- Priority detection based on keywords
- REST API included for developers who want to integrate
- Local storage persistence
- Keyboard shortcuts for power users

Technical Details
- Built with Electron for desktop
- Progressive Web App for web and mobile
- Express.js API server with OpenAPI documentation
- No external dependencies required for basic use
- Open source with MIT license
