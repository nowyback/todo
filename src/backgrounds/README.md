# Backgrounds Folder

**LOCAL FILES ONLY** - No internet URLs supported!

## Location
This folder is located at: `src/backgrounds/`

## Supported Formats
- JPG
- PNG  
- WebP
- GIF

## How to Use
1. Place images in this folder ONLY
2. In Todoodle, press Ctrl+B
3. Use the file browser to select images
4. Click "Browse" to apply background

## Examples
- `dark-theme.jpg` (for dark mode)
- `light-theme.jpg` (for light mode)
- `nature.jpg` (works for both)

## Rules
- ✅ Local files ONLY
- ❌ NO internet URLs
- ❌ NO external links
- ✅ Place files in this folder

## Tips
- Use high-resolution images for best results
- Images should be at least 1920x1080
- Darker images work better for dark mode
- Lighter images work better for light mode
- Keep images under 10MB for performance
