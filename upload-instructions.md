# Distribution Instructions

## GitHub Release Setup

### Step 1: Upload to File Hosting
1. Upload `dist/Todoodle Setup 0.9.0.exe` to:
   - Google Drive (share link)
   - Dropbox (share link)
   - WeTransfer (temporary)
   - Your own website

### Step 2: Create GitHub Release
1. Go to: https://github.com/nowyback/todoodle/releases
2. Create new release with tag: `v0.9.1`
3. Add download link in description:
   ```
   ## Download Windows
   🔗 [Download Todoodle Setup 0.9.0.exe](YOUR_FILE_LINK)
   
   ## Requirements
   - Windows 10 or later
   - 84MB download size
   
   ## Installation
   1. Download the installer
   2. Run the setup file
   3. Follow installation wizard
   4. Search "Todoodle" in Start Menu
   ```

### Alternative: Use GitHub Pages
1. Enable GitHub Pages in repository settings
2. Create `gh-pages` branch
3. Upload installer to branch
4. Link directly: `https://nowyback.github.io/todoodle/Todoodle%20Setup%200.9.0.exe`

## File Hosting Options
- **Google Drive**: Free, 15GB storage
- **Dropbox**: Free, 2GB storage
- **WeTransfer**: Free, 2GB per transfer
- **GitHub Pages**: Free, 1GB storage
