🖥️ Todoodle Server Setup

For advanced users who want to run Todoodle with a backend server:

  • Node.js 16+ installed
  • npm package manager
  • Terminal/command prompt access
  • Local network access

Quick Setup:

# Clone the repository
git clone https://github.com/yourusername/todoodle.git
cd todoodle

# Install dependencies
npm install

# Start the server
npm run server

# Start the app
npm start

Server will run on http://localhost:3001 by default
